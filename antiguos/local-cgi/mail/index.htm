Este es un ejemplo de formulario para utilizar con el FormMail. A continuación le mostraremos como customizarlo. Así es como esta configurado actualmente su FormMail.cgi, esto datos se pueden cambiar y ajustar a sus necesidades. Todos lo que se encuentre entre <!-- y --> es un comentario dentro de su codigo HTML. Los campos que son type=hidden son campos ocultos y para modificarlos debe entrar en modo de vista de código html para editarlos. Los nombres de estos campos listados mas abajo se deben respetar.
<!-- Lista de cuentas de email a las cuales se le envia el email separada por comas-->
<input type=hidden name="recipient" value="info@lelen.com.ar">
<!-- Asunto del mensaje -->
<input type=hidden name="subject" value="Consulta desde la web">
<!-- Página a la cual se quiere redireccionar al enviar el formulario -->
<input type=hidden name="redirect" value="gracias.htm">
<!-- Campos requeridos -->
<input type=hidden name="required" value="email,realname,telefono">
<!-- Ordenamiento de los campos en el mail enviado -->
<input type=hidden name="sort" value="order:realname,email,telefono,direccion,Contacto,Rubro">
<!-- Página de error customizada si falta algun dato requerido con el cual es llenado el formulario-->
<!-- Sí elimina este campo se mostrará la página de error del propio FormMail.cgi el cual le indica cuales son los campos requeridos que no fueron llenado ->
